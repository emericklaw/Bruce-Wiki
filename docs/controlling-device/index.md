# User Interface

Depending on your device the way you interact with Bruce may differ.

The common methods include:

* M5Stack Sticks
    * Side buttons - left/right next/previous
        * Long hold previous button to power off device
    * M5 button - enter/select
* T-Embed
    * Encoder wheel - left/right next/previous
    * Encoder button - enter/select
    * Side button - back
        * Long hold to put device into deep sleep
* M5Stack Cardputer
    * Direction buttons - left/right next/previous
    * Enter - enter/select
    * Esc - back
    * [Keyboard Shortcuts](keyboard-shortcuts.md)
